# Introduction

## Description

Task manager web app. Manage your TODO lists easily with this web app.

## Developer Notes

This project uses AWS Amplify for backend resources and hosting. Do the following to get started:

### Create resources

```
amplify init
amplify add auth
amplify add hosting
amplify push
amplify publish
```

### Tear down

```
amplify delete
```
